# Claude Code Onboarding

Presentation slides about CLAUDE.md files for AI coding assistants.

## Usage

```bash
# Generate HTML
npx @marp-team/marp-cli slides.md -o slides.html

# Live preview
npx @marp-team/marp-cli slides.md -w
```
