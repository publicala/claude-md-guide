# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

Presentation slides about CLAUDE.md files for AI coding assistants. Uses Marp (Markdown Presentation Ecosystem).

## Commands

- `npx @marp-team/marp-cli slides.md -o slides.html` - Generate HTML slides
- `npx @marp-team/marp-cli slides.md --pdf -o slides.pdf` - Generate PDF
- `npx @marp-team/marp-cli slides.md -w` - Watch mode with live reload

## Structure

- `slides.md` - Presentation content with Marp frontmatter and custom CSS
- `slides.html` - Generated output (regenerate after editing slides.md)
